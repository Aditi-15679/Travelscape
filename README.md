# Beyond Horizons 🌏✈️

**Beyond Horizons** is a modern, responsive travel landing page designed to inspire and guide travelers. This project showcases a curated collection of beautiful destinations worldwide, blending sleek aesthetics with smooth interactivity.

---

## 🌟 Features

* **Responsive Design:** Optimized for desktop, tablet, and mobile devices.
* **Hero Section:** A visually striking intro with a clear Call-to-Action (CTA).
* **Top Destinations:** A clean gallery featuring local imagery and enticing descriptions.
* **High Performance:** Uses local assets for fast loading and offline compatibility.
* **Modern UI:** Built with **Google Fonts (Poppins)** for a professional, readable feel.
* **Smooth Navigation:** Fixed header for effortless browsing across sections.

---

## 🛠️ Technologies Used

* **HTML5** – Semantic structure and layout.
* **CSS3** – Custom styling, Flexbox/Grid, and responsive media queries.
* **Google Fonts** – Enhanced typography for better readability.

## 🚀 How to Run

* Clone the repository:  
  `git clone <your-github-repo-url>`

* Navigate to the folder:  
  `cd Travelscape`

* Launch the site:  
  Simply open `index.html` in any modern web browser.  

> Note: No server or build tools are required. It’s ready to go out of the box!
